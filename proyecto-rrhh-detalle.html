Descripción del Proyecto

RRHH es un sistema integral de administración de recursos humanos desarrollado durante el sexto semestre en la materia de Programación II. El sistema permite gestionar de manera eficiente la información de empleados, sus datos personales y contratos laborales.

Este proyecto destacó por la implementación y consumo de APIs REST, demostrando competencias en integración de servicios, comunicación entre sistemas y desarrollo de software distribuido. El proyecto también enfatizó el trabajo en equipo a mayor escala, aplicando metodologías ágiles y buenas prácticas de desarrollo colaborativo.

El sistema facilita la gestión completa del ciclo de vida de los empleados, desde el proceso de contratación hasta la administración de sus datos personales y contractuales, proporcionando una solución moderna y escalable para departamentos de recursos humanos.

Características principales:

  • Gestión completa de información de empleados
  • Administración de datos personales y contacto
  • Gestión de contratos laborales y documentos
  • Integración con APIs REST para servicios externos
  • Consumo de APIs para validación de datos
  • API propia para acceso a información de empleados
  • Autenticación y autorización de usuarios
  • Búsqueda y filtrado avanzado de empleados
  • Generación de reportes y documentos
  • Historial de cambios y auditoría
  • Interfaz intuitiva para administradores
  • Arquitectura escalable y mantenible

Código del Proyecto

Ejemplos de implementación de APIs y consumo de servicios: 

// Ejemplo de endpoint API REST para obtener empleados
@RestController
@RequestMapping("/api/empleados")
public class EmpleadoController {
    
    @Autowired
    private EmpleadoService empleadoService;
    
    // GET /api/empleados
    @GetMapping
    public ResponseEntity<List<Empleado>> obtenerEmpleados() {
        List<Empleado> empleados = empleadoService.obtenerTodos();
        return ResponseEntity.ok(empleados);
    }
    
    // GET /api/empleados/{id}
    @GetMapping("/{id}")
    public ResponseEntity<Empleado> obtenerEmpleado(
            @PathVariable Long id) {
        Empleado empleado = empleadoService.obtenerPorId(id);
        if (empleado == null) {
            return ResponseEntity.notFound().build();
        }
        return ResponseEntity.ok(empleado);
    }
    
    // POST /api/empleados
    @PostMapping
    public ResponseEntity<Empleado> crearEmpleado(
            @RequestBody Empleado empleado) {
        Empleado nuevoEmpleado = empleadoService.crear(empleado);
        return ResponseEntity.status(HttpStatus.CREATED)
                .body(nuevoEmpleado);
    }
}
// Cliente para consumir API externa de validación
import org.springframework.web.client.RestTemplate;

public class ValidacionService {
    private final String API_URL = "https://api.validacion.com/v1";
    private RestTemplate restTemplate;
    
    public ValidacionService() {
        this.restTemplate = new RestTemplate();
    }
    
    // Validar CURP usando API externa
    public boolean validarCURP(String curp) {
        try {
            String url = API_URL + "/curp/validar?curp=" + curp;
            ResponseEntity<ValidacionResponse> response = 
                restTemplate.getForEntity(url, ValidacionResponse.class);
            
            return response.getBody().isValido();
        } catch (Exception e) {
            System.err.println("Error al validar CURP: " + e.getMessage());
            return false;
        }
    }
    
    // Obtener datos de empleado desde API externa
    public EmpleadoExterno obtenerDatosExternos(String id) {
        String url = API_URL + "/empleados/" + id;
        return restTemplate.getForObject(url, EmpleadoExterno.class);
    }
}
