Descripción del Proyecto

RAC (Reconocimiento de Acceso Controlado) es un sistema integral desarrollado para mejorar la seguridad, gestión del espacio y recursos en los laboratorios del Instituto Tecnológico de Toluca. El sistema combina tecnología RFID y biométrica para controlar las entradas y salidas de estudiantes de manera eficiente y segura.

El proyecto evolucionó a lo largo de varios semestres, incorporando nuevas funcionalidades como una interfaz web para administración remota, notificaciones de inasistencias, gestión de horarios y supervisión en tiempo real de las actividades en los laboratorios del Edificio T.

Este sistema demuestra la integración de hardware (lectores RFID, sensores biométricos) con software de escritorio (Java) y aplicaciones web (PHP), conectándose a una base de datos MySQL para el almacenamiento y gestión de información.

Características principales:

  • Control de acceso mediante tarjetas RFID
  • Reconocimiento biométrico para mayor seguridad
  • Interfaz web para administración remota
  • Gestión de usuarios, horarios y permisos
  • Registro automático de entradas y salidas
  • Sistema de notificaciones de inasistencias
  • Supervisión en tiempo real de actividades
  • Análisis y reportes de acceso
  • Integración con base de datos MySQL
  • Aplicación de escritorio en Java para control local
