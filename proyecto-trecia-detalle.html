TRECIA

Sistema de gestión hospitalaria desarrollado en JAVA para la administración integral de hospitales

Java POO Swing Sistema de Gestión Tercer Semestre

Descripción del Proyecto

TRECIA es un sistema de gestión hospitalaria desarrollado en JAVA utilizando Programación Orientada a Objetos (POO). Este proyecto fue creado durante el tercer semestre de la carrera y tiene como objetivo facilitar la administración de diferentes aspectos de un hospital, desde la gestión de pacientes hasta el control de recursos médicos.

El sistema implementa los principios fundamentales de POO como encapsulamiento, herencia, polimorfismo y abstracción, demostrando la capacidad de diseñar y desarrollar software robusto y escalable para entornos críticos como el sector salud.

Características principales:

  • Gestión de pacientes y expedientes médicos
  • Control de citas y consultas médicas
  • Administración de personal médico y administrativo
  • Gestión de inventario de medicamentos y equipos
  • Interfaz gráfica desarrollada con Java Swing
  • Arquitectura basada en principios de POO
  • Sistema de validación de datos
  • Manejo de excepciones y errores

Código del Proyecto

A continuación se muestra un ejemplo de la estructura del código utilizando POO: 

// Clase principal Paciente - Ejemplo de encapsulamiento
public class Paciente {
    private String nombre;
    private String curp;
    private String fechaNacimiento;
    private String telefono;
    
    // Constructor
    public Paciente(String nombre, String curp, 
                          String fechaNacimiento, String telefono) {
        this.nombre = nombre;
        this.curp = curp;
        this.fechaNacimiento = fechaNacimiento;
        this.telefono = telefono;
    }
    
    // Getters y Setters
    public String getNombre() {
        return nombre;
    }
    
    public void setNombre(String nombre) {
        this.nombre = nombre;
    }
    
    // Método para mostrar información
    public String toString() {
        return "Paciente: " + nombre + ", CURP: " + curp;
    }
}
// Ejemplo de herencia - Médico hereda de Personal
public class Personal {
    protected String nombre;
    protected String idEmpleado;
    
    public Personal(String nombre, String idEmpleado) {
        this.nombre = nombre;
        this.idEmpleado = idEmpleado;
    }
}

public class Medico extends Personal {
    private String especialidad;
    private String cedula;
    
    public Medico(String nombre, String idEmpleado, 
                      String especialidad, String cedula) {
        super(nombre, idEmpleado);
        this.especialidad = especialidad;
        this.cedula = cedula;
    }
    
    public void atenderPaciente(Paciente paciente) {
        System.out.println("Dr. " + nombre + " atendiendo a " + 
                          paciente.getNombre());
    }
}
