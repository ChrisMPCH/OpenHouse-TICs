Descripción del Proyecto

CAC One es una aplicación móvil innovadora que utiliza técnicas de Inteligencia Artificial y Procesamiento de Lenguaje Natural (NLP) para analizar el sentimiento de comentarios relacionados con General Motors (GM). El sistema es capaz de determinar si un comentario es positivo, negativo o neutro, proporcionando insights valiosos para la toma de decisiones empresariales.

Desarrollado durante el séptimo semestre en la materia de Desarrollo de Aplicaciones Móviles, este proyecto demuestra la integración de tecnologías de IA con desarrollo móvil, aplicando modelos de machine learning para análisis de sentimiento en tiempo real.

El sistema procesa texto en lenguaje natural, identifica patrones emocionales y proporciona métricas cuantitativas sobre la percepción de la marca, permitiendo a las empresas monitorear su reputación y responder proactivamente a las opiniones de los clientes.

Características principales:

  • Análisis de sentimiento en tiempo real

Ejemplos de implementación de análisis de sentimiento: 

// Servicio de análisis de sentimiento
import 'package:http/http.dart' as http;
import 'dart:convert';

class SentimentAnalysisService {
  final String apiUrl = 'https://api.sentiment.com/v1/analyze';
  
  Future<SentimentResult> analyzeSentiment(String text) async {
    try {
      final response = await http.post(
        Uri.parse(apiUrl),
        headers: {
          'Content-Type': 'application/json',
          'Authorization': 'Bearer $apiKey',
        },
        body: jsonEncode({
          'text': text,
          'language': 'es',
        }),
      );
      
      if (response.statusCode == 200) {
        final data = jsonDecode(response.body);
        return SentimentResult.fromJson(data);
      } else {
        throw Exception('Error al analizar sentimiento');
      }
    } catch (e) {
      throw Exception('Error: $e');
    }
  }
}

class SentimentResult {
  final String sentiment; // positive, negative, neutral
  final double confidence;
  final double score;
  
  SentimentResult({
    required this.sentiment,
    required this.confidence,
    required this.score,
  });
  
  factory SentimentResult.fromJson(Map<String, dynamic> json) {
    return SentimentResult(
      sentiment: json['sentiment'],
      confidence: json['confidence'].toDouble(),
      score: json['score'].toDouble(),
    );
  }
}
// Widget para mostrar resultado del análisis
class SentimentResultWidget extends StatelessWidget {
  final SentimentResult result;
  
  SentimentResultWidget({required this.result});
  
  @override
  Widget build(BuildContext context) {
    Color sentimentColor = result.sentiment == 'positive' 
        ? Colors.green 
        : result.sentiment == 'negative' 
            ? Colors.red 
            : Colors.orange;
    
    return Card(
      child: Padding(
        padding: EdgeInsets.all(16.0),
        child: Column(
          children: [
            Text(
              'Sentimiento: ${result.sentiment.toUpperCase()}',
              style: TextStyle(
                fontSize: 20,
                fontWeight: FontWeight.bold,
                color: sentimentColor,
              ),
            ),
            SizedBox(height: 10),
            LinearProgressIndicator(
              value: result.confidence,
              backgroundColor: Colors.grey[300],
              valueColor: AlwaysStoppedAnimation<Color>(sentimentColor),
            ),
            SizedBox(height: 5),
            Text('Confianza: ${(result.confidence * 100).toStringAsFixed(1)}%'),
          ],
        ),
      ),
    );
  }
}
