Descripción del Proyecto

DeVstyle es una plataforma e-commerce innovadora diseñada específicamente para la comunidad tech. La tienda ofrece ropa cómoda y funcional, así como accesorios personalizados que reflejan la identidad y valores de los profesionales de la tecnología.

El proyecto fue desarrollado utilizando la arquitectura Modelo-Vista-Controlador (MVC), lo que permite una separación clara de responsabilidades, facilitando el mantenimiento y la escalabilidad del sistema. La plataforma está optimizada para ofrecer una experiencia de compra intuitiva y segura.

Este proyecto demuestra competencias en desarrollo web full-stack, gestión de proyectos, diseño de interfaces de usuario (UX/UI) y negocios electrónicos, siendo desarrollado a lo largo de varios semestres como parte de diferentes materias incluyendo Ingeniería de Software, Taller de Ingeniería de Software y Desarrollo Emprendedor.

Características principales:

  • Catálogo de productos con categorías y filtros avanzados
  • Sistema de carrito de compras y checkout
  • Gestión de usuarios y autenticación
  • Panel de administración para gestión de productos
  • Procesamiento de pagos seguro
  • Gestión de inventario en tiempo real
  • Diseño responsive para dispositivos móviles
  • Interfaz intuitiva y moderna
  • Sistema de búsqueda y recomendaciones
  • Gestión de pedidos y envíos
  • Arquitectura MVC para código mantenible

Código del Proyecto

Ejemplos de la arquitectura MVC implementada: 

// Modelo - Producto.php
<?php
class Producto {
    private $db;
    
    public function __construct($database) {
        this->db = $database;
    }
    
    public function obtenerProductos() {
        $query = "SELECT * FROM productos WHERE activo = 1";
        return this->db->query($query)->fetchAll();
    }
    
    public function obtenerProductoPorId($id) {
        $query = "SELECT * FROM productos WHERE id = ? AND activo = 1";
        $stmt = this->db->prepare($query);
        $stmt->execute([$id]);
        return $stmt->fetch();
    }
    
    public function agregarProducto($datos) {
        $query = "INSERT INTO productos (nombre, precio, descripcion, categoria_id) 
                VALUES (?, ?, ?, ?)";
        $stmt = this->db->prepare($query);
        return $stmt->execute([
            $datos['nombre'],
            $datos['precio'],
            $datos['descripcion'],
            $datos['categoria_id']
        ]);
    }
}
?>
// Controlador - ProductoController.php
<?php
require_once 'models/Producto.php';

class ProductoController {
    private $productoModel;
    
    public function __construct($db) {
        this->productoModel = new Producto($db);
    }
    
    public function index() {
        $productos = this->productoModel->obtenerProductos();
        require_once 'views/productos/index.php';
    }
    
    public function ver($id) {
        $producto = this->productoModel->obtenerProductoPorId($id);
        if (!$producto) {
            header('Location: /productos');
            exit;
        }
        require_once 'views/productos/ver.php';
    }
}
?>
