LUXURY TRIP

Sistema de gestión de viajes cortos desarrollado en JAVA para la administración de viajes entre diferentes partes del mundo

Java Estructuras de Datos Algoritmos Sistema de Gestión Quinto Semestre

Descripción del Proyecto

LUXURY TRIP es un sistema de gestión de viajes desarrollado en JAVA que permite administrar y organizar viajes cortos entre diferentes destinos alrededor del mundo. El proyecto fue desarrollado durante el quinto semestre como parte de la materia de Estructuras de Datos, demostrando la aplicación práctica de diferentes estructuras de datos y algoritmos en un contexto real.

El sistema implementa estructuras de datos eficientes como listas enlazadas, árboles, grafos y tablas hash para optimizar las operaciones de búsqueda, filtrado y gestión de información de viajes, destinos, clientes y reservaciones.

Este proyecto demuestra competencias en el diseño e implementación de algoritmos eficientes, manejo de estructuras de datos complejas, y desarrollo de software orientado a objetos para resolver problemas del mundo real en el sector turístico.

Características principales:

  • Gestión de destinos turísticos y rutas
  • Administración de clientes y reservaciones
  • Búsqueda eficiente de viajes por criterios múltiples
  • Cálculo de rutas óptimas entre destinos
  • Gestión de disponibilidad y precios
  • Implementación de estructuras de datos avanzadas
  • Algoritmos de ordenamiento y búsqueda optimizados
  • Interfaz de usuario intuitiva
  • Validación de datos y manejo de excepciones
  • Persistencia de datos en archivos

// Clase Viaje con estructura de datos
public class Viaje {
    private String destino;
    private String origen;
    private double precio;
    private Date fechaSalida;
    private int duracion;
    
    public Viaje(String origen, String destino, 
                  double precio, Date fechaSalida, int duracion) {
        this.origen = origen;
        this.destino = destino;
        this.precio = precio;
        this.fechaSalida = fechaSalida;
        this.duracion = duracion;
    }
    
    // Getters y Setters
    public String getDestino() {
        return destino;
    }
    
    public double getPrecio() {
        return precio;
    }
}
// Gestor de viajes usando HashMap para búsqueda eficiente
import java.util.*;

public class GestorViajes {
    private HashMap<String, List<Viaje>> viajesPorDestino;
    private List<Viaje> todosLosViajes;
    
    public GestorViajes() {
        viajesPorDestino = new HashMap<>();
        todosLosViajes = new ArrayList<>();
    }
    
    public void agregarViaje(Viaje viaje) {
        todosLosViajes.add(viaje);
        String destino = viaje.getDestino();
        
        if (!viajesPorDestino.containsKey(destino)) {
            viajesPorDestino.put(destino, new ArrayList<>());
        }
        viajesPorDestino.get(destino).add(viaje);
    }
    
    // Búsqueda O(1) por destino usando HashMap
    public List<Viaje> buscarPorDestino(String destino) {
        return viajesPorDestino.getOrDefault(destino, 
                                         new ArrayList<>());
    }
    
    // Ordenamiento por precio usando Collections.sort
    public List<Viaje> obtenerViajesOrdenadosPorPrecio() {
        List<Viaje> ordenados = new ArrayList<>(todosLosViajes);
        Collections.sort(ordenados, (v1, v2) -> 
            Double.compare(v1.getPrecio(), v2.getPrecio()));
        return ordenados;
    }
}
